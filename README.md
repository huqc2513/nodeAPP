nodeApp
====
简介
----
这是一个简单的商城app，使用了nodejs、koa2,koa-mysql,koa-session,multer等功能模块。<br>

由于前后端分离开发，所以后端的主要任务是向前端传输json。<br>

前端的是基于vue.js来实现，代码可以点击这里访问到。<br>

##目前实现的功能有：<br>

##注册<br>
##登陆/登出<br>
##添加商品<br>

##添加购物车并结算<br>
##项目运行<br>

> 安装依赖<br>
 ##npm install<br>

##npm run dev<br>
