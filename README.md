nodeApp
====
简介
----
这是一个简单的商城app，后端使用了nodejs、koa2,koa-mysql,multer,cors等功能模块。<br>

由于前后端分离，所以后端的主要任务是向前端传输json。<br>

前端的是基于vue.js element-ui,mint-ui  来实现<br>


##目前实现的功能有：<br>

##注册<br>
##登陆/登出<br>
##添加商品<br>
##删除商品<br>
##搜索商品<br>
##下单，添加,删除订单<br>
##添加购物车并结算<br>
##项目运行<br>
